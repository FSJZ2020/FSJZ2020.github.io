Chrome快捷键

发表于 2021-10-10 阅读次数： 本文字数： 1.6k 阅读时长 ≈ 1 分钟

选项卡和窗口快捷方式

含义 快捷键
打开新窗口 Ctrl+n
在隐身模式下打开新窗口 Ctrl+Shift+n
打开并跳转到新选项卡 Ctrl+t
按照关闭的顺序重新打开之前的选项卡 Ctrl+Shift+t
跳转到下一个选项卡 Ctrl+Tab或Ctrl+PgDn
跳转到上一个选项卡 Ctrl+Shift+Tab或Ctrl+PgUp
跳到特定的选项卡 Ctrl+1~8
跳转到最后一个选项卡 Ctrl+9
在当前选项卡打开主页 Alt+Home
在当前选项卡打开浏览历史记录上一页 Alt+←
在当前选项卡打开浏览历史记录下一页 Alt+→
关闭当前选项卡 Ctrl+F4或Ctrl+w
关闭当前窗口 Alt+F4或Ctrl+Shift+w

功能快捷方式

含义 快捷键
打开chrome菜单 Alt+e或Alt+f
显示或隐藏书签栏 Ctrl+Shift+b
打开书签管理器 Ctrl+Shift+o
在新选项卡打开历史记录页面 Ctrl+h
在新选项卡打开下载页面 Ctrl+j
打开chrome任务管理器 Shift+Esc
将焦点设置在chrome工具栏的第一项上 Shift+Alt+t
将焦点设置在chrome工具栏最右边的项目上 F10
将焦点切换到未聚焦的对话框和所有工具栏 F6
打开查找栏搜索当前页面 Ctrl+f或F3
跳转到查找栏搜索的下一个匹配项 Ctrl+g
跳转到查找栏搜索的上一个匹配项 Ctrl+Shift+g
打开开发者工具 F12或Ctrl+Shift+j
打开清除浏览数据选项 Ctrl+Shift+Delete
在新标签页中打开 chrome 帮助中心 F1
以不同的用户登录或以访客身份浏览 Ctrl+Shift+m
打开插入符号浏览 F7
打开反馈表 Alt+Shift+i

地址栏快捷方式

含义 快捷键
添加www.和.com到站点名称，并在当前选项卡中打开它 输入站点名称+ Ctrl + Enter
打开一个新标签并执行搜索 输入搜索词+Alt+Enter
跳转到地址栏 Ctrl+l或Alt+d或F6
从页面上的任何位置搜索 Ctrl+k或Ctrl+e
从地址栏中删除预测 选中该项+Shift+Delete
将光标移动到地址栏 Ctrl+F5

网页快捷方式

含义 快捷键
打印当前页面 Ctrl + p
保存当前页面 Ctrl + s
重新加载当前页面 F5或Ctrl+r
重新加载当前页面，忽略缓存内容 Shift+F5或Ctrl+Shift+r
停止页面加载 Esc
向前浏览可点击的项目 Tab
浏览向后移动的可点击项目 Shift+Tab
在 Chrome 中打开计算机中的文件 Ctrl+o+选择一个文件
显示当前页面的不可编辑的 HTML 源代码 Ctrl+u
将您当前的网页保存为书签 Ctrl+d
将所有打开的标签保存为新文件夹中的书签 Ctrl+Shift+d
打开或关闭全屏模式 F11
放大页面上的所有内容 Ctrl++
缩小页面上的所有内容 Ctrl+-
将页面上的所有内容恢复为默认大小 Ctrl+0
向下滚动网页，一次一个屏幕 空格或PgDn
向上滚动网页，一次一个屏幕 Shift+空格或PgUp
转到页面的顶部 Home
转到页面底部 End
在页面上水平滚动 Shift+滚动鼠标滚轮
将光标移动到文本字段中上一个单词的开头 Ctrl+←
将光标移动到下一个单词 Ctrl+→
删除文本字段中的上一个单词 Ctrl+Backspace

鼠标快捷键

含义 快捷键
在当前选项卡中打开链接 将链接拖到选项卡
在新的选项卡中打开链接 Ctrl+单击链接
在新的选项卡中打开链接，并跳转 Ctrl+Shift+单击链接
打开一个链接，并跳转（仅限鼠标） 将链接拖到标签栏的空白区域
在新窗口中打开链接 Shift +单击链接
在新窗口中打开选项卡（仅限鼠标） 将标签拖出标签条
将选项卡移动到当前窗口（仅限鼠标） 将选项卡拖到现有窗口中
将选项卡返回到其原始位置 拖动时按Esc
将当前网页保存为书签 将网址拖到书签栏
下载链接的目标 Alt+单击链接
显示您的浏览历史 右键单击“返回” 或单击并按住“返回”
右键单击“下一步” 或单击并按住“下一步”
在最大化和窗口模式之间切换 双击标签条的空白区域
放大页面 Ctrl+向上滚动
缩小页面 Ctrl+向下滚动
